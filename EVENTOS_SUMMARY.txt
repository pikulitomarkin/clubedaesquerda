CAMPOS COMUNS (Todos os eventos)
├─ Nome (texto, obrigatório)
├─ Organizador (texto, obrigatório)
├─ Data (date picker, obrigatório)
├─ Horário (time picker, obrigatório)

├─ Imagem de Capa (upload, 5MB max, preview)
└─ Validação de preenchimento

TIPO 1: PRESENCIAL 📍
├─ Local (obrigatório)
│  └─ Ex: "Praça da República"
├─ Endereço Completo (obrigatório)
│  └─ Ex: "Rua X, 123, Bairro Y, Cidade Z, Estado"
├─ Capacidade (opcional, número)
│  └─ Número máximo de participantes
└─ Validações: local e endereço obrigatórios

TIPO 2: ONLINE 💻
├─ Plataforma (select obrigatório)
│  ├─ Google Meet
│  ├─ Zoom
│  ├─ Jitsi
│  └─ Outra
├─ Link da Reunião (URL obrigatório)
│  └─ Validação de URL
├─ Senha ou Token (opcional)
│  └─ Para acesso restrito
└─ Validações: link URL válida

TIPO 3: CLUBE (Roda de Conversa) 💬
├─ Local (obrigatório)
├─ Recorrência (select obrigatório)
│  ├─ Nenhuma (evento único)
│  ├─ Semanal
│  ├─ Quinzenal
│  └─ Mensal
├─ Data Final (opcional)
│  └─ Quando para a recorrência
├─ Tema (opcional)
│  └─ Ex: "Feminismo e Trabalho"
├─ Objetivos (opcional, 200 chars)
├─ Facilitadores (dinâmico)
│  ├─ Pelo menos 1 obrigatório
│  ├─ Adicionar mais com botão +
│  ├─ Remover com botão X
│  └─ Validação: nomes não vazios
└─ Próximos Encontros (gerado automaticamente)

TIPO 4: ANÁLISE (Círculo de Estudos) 📚
├─ Local (obrigatório)
├─ Recorrência (select obrigatório)
│  ├─ Nenhuma
│  ├─ Semanal
│  ├─ Quinzenal
│  └─ Mensal
├─ Data Final (opcional)
├─ Metodologia (opcional, 200 chars)
│  └─ Ex: "Leitura prévia + debate"
├─ Facilitadores (dinâmico, obrigatório)
├─ Textos para Preparar (dinâmico, opcional)
│  ├─ Adicionar com botão +
│  ├─ Remover com botão X
│  └─ Ex: "Marx - O Capital, cap. 1"
├─ Questões-chave (dinâmico, opcional)
│  └─ Ex: "Qual o papel dos intelectuais?"
└─ Próximos Encontros (gerado automaticamente)

🎯 INTERAÇÕES
═══════════════════════════════════════════════════════════════════════════════

CRIAR NOVO EVENTO
1. Usuário clica "Novo [Tipo]" (qualquer botão)
2. setTipoSelecionado(tipo)
3. setModalAberto(true)
4. Modal abre com formulário correto
5. Preenche campos (validação inline)
6. Clica "Criar Evento" ou "Cancelar"
7. Se submeter:
   ├─ handleSubmitEvento(data)
   ├─ setIsLoading(true)
   ├─ await delay(1000)
   ├─ Novo evento criado com ID único
   ├─ setEventos([...eventos, novoEvento])
   ├─ setModalAberto(false)
   ├─ showNotificacao("X criado com sucesso!")
   ├─ setIsLoading(false)
   └─ Lista atualiza
8. Se cancelar:
   ├─ handleFecharModal()
   ├─ Modal fecha
   └─ Sem salvar

BUSCAR/FILTRAR
1. Usuário digita na busca
2. setFiltros({ busca: valor })
3. eventosFiltrados recalculados
4. Lista atualiza em tempo real
5. Usuário seleciona tipo
6. setFiltros({ tipo: valor })
7. Lista filtra por tipo
8. Checkbox "Apenas ativos"
9. setFiltros({ apenas_ativos: true/false })
10. Lista filtra por ativo
11. Botão "Limpar filtros"
12. Reset para defaults

ADICIONAR FACILITADOR/TEXTO/QUESTÃO
1. Clica "Adicionar"
2. addItem(field)
3. Nova linha vazia adicionada
4. Digita conteúdo
5. handleArrayChange(field, index, value)
6. Array atualizado
7. Clica X para remover
8. removeItem(field, index)
9. Item removido do array
