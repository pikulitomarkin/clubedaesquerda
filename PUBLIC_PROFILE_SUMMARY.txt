🎯 ESTADOS DE USUÁRIO
═══════════════════════════════════════════════════════════════════════════════

VENDO PERFIL DE OUTRO USUÁRIO
├─ isSelf = false
├─ Vê botão "+ Seguir" / "✓ Seguindo"
├─ NÃO vê botão "Editar Perfil"
├─ NÃO vê toggle de homenagens
├─ NÃO pode deletar homenagens
├─ Vê todas as homenagens públicas
└─ Pode clicar em rodas/eventos

VENDO SEU PRÓPRIO PERFIL
├─ isSelf = true
├─ NÃO vê botão "+ Seguir"
├─ Vê botão "✏️ Editar Perfil"
├─ Vê toggle "👁️ Visível/Oculto" em homenagens
├─ Pode deletar homenagens individuais
├─ Vê X/Y homenagens (selecionadas/total)
├─ Quando oculto: vê "Homenagens Ocultas" com conte total
└─ Pode compartilhar link de homenagem

🔄 FLUXOS
═══════════════════════════════════════════════════════════════════════════════

CARREGAR PERFIL
1. PublicProfilePage renderiza com userId
2. useEffect inicia carregamento
3. simulação de 500ms de delay
4. Profile state atualizado
5. Componentes renderizam com dados

SEGUIR/DEIXAR DE SEGUIR
1. Usuário clica "+ Seguir"
2. onFollowToggle() chamado
3. isFollowing = true
4. followers += 1
5. Botão muda para "✓ Seguindo"
6. POST /api/profiles/:userId/follow (backend)

TOGGLE HOMENAGENS
1. Proprietário clica "👁️ Visível" / "Oculto"
2. onToggleVisibility(!showHomenagens)
3. showHomenagens state muda
4. Profile re-renderiza
5. Mostra/oculta homenagens
6. PATCH /api/profiles/:userId/honor-visibility (backend)

DELETAR HOMENAGEM
1. Proprietário clica X em homenagem
2. onDeleteHomenagem(id) chamado
3. Filter out da homenagem
4. Contador atualizado
5. DELETE /api/homenagens/:id (backend)
